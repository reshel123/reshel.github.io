## ✨ 主要功能

### 🤖 AI智能助手
- **多模型支持**: 支持GPT-4、Claude、Gemini、Deepseek等多种AI模型
- **多标签对话**: 每个模型独立的对话窗口，可自由切换
- **实时响应**: 基于ChatAnywhere API，无需VPN即可使用
- **中英文界面**: 支持语言切换，适合国内外用户

### 📦 软件下载中心
- **分类管理**: Windows应用、安卓应用、媒体文件
- **直接下载**: 支持游戏安装包、压缩包、图片、视频等
- **预览功能**: 媒体文件支持在线预览
- **外部链接**: 所有文件均使用真实外部下载链接

### 🎮 小游戏中心
- **贪吃蛇**: 经典贪吃蛇游戏，支持键盘和触摸控制
- **俄罗斯方块**: 经典俄罗斯方块，支持旋转和移动
- **Flappy Bird**: 小鸟飞行游戏，点击开始后需点击屏幕启动
- **数字华容道**: 滑块拼图游戏，将数字排列成正确顺序
- **记忆翻牌**: 考验记忆力的卡片配对游戏
- **打砖块**: 经典打砖块游戏，支持键盘和鼠标控制

### 🔒 隐藏功能
- **秘密入口**: 在软件下载页面右上角
- **密码保护**: 需要输入正确暗号才能进入
- **视频播放**: 全屏CSS模拟视频播放器
- **神秘体验**: 为网站增添趣味性

### 🌤️ 天气显示
- **实时天气**: 基于和风天气API
- **位置获取**: 自动获取用户位置
- **城市显示**: 显示当前城市名称
- **缓存优化**: 减少重复API调用

### 🎵 音乐播放器
- **在线播放**: 支持外部MP3链接播放
- **控制功能**: 播放/暂停、音量调节
- **美观设计**: 与网站整体风格一致

## 🎯 使用指南

### 主页面功能
1. **语言切换**: 右上角中英文切换按钮
2. **天气显示**: 左上角实时天气信息
3. **音乐播放**: 右上角音乐播放器
4. **功能导航**: 点击卡片或按钮进入相应功能

### AI聊天使用
1. 点击"AI助手"进入聊天页面
2. 点击"+"按钮添加新对话
3. 选择AI模型开始对话
4. 支持多标签切换不同模型

### 软件下载使用
1. 点击"下载中心"进入软件页面
2. 选择分类查看不同软件
3. 点击"下载"按钮直接下载
4. 媒体文件支持预览功能

### 小游戏使用
1. 在软件下载页面点击"🎮 小游戏"
2. 选择想要玩的游戏
3. 点击"开始游戏"开始
4. 查看游戏说明和操作方式

### 隐藏功能使用
1. 在软件下载页面找到"🔒 隐藏入口"
2. 输入正确暗号（小写英文）
3. 进入隐藏页面观看视频

## 🎨 自定义

### 添加新软件
在 `software-library.html` 中的 `softwareData` 数组添加：
```javascript
{
    id: "新软件ID",
    name: "软件名称",
    category: "Windows应用",
    description: "软件描述",
    size: "文件大小",
    downloadUrl: "下载链接"
}
```

### 添加新游戏
1. 创建新的游戏HTML文件
2. 在 `game-page.html` 中添加游戏卡片
3. 实现游戏逻辑和界面

### 修改样式
- 主要样式在HTML文件的 `<style>` 标签中
- 支持CSS3高级特性
- 响应式设计适配不同设备
