## 🎯 使用指南

### 主页面功能
1. **智能加载**: 90秒倒计时或真实加载完成
2. **语言切换**: 右上角中英文切换按钮
3. **天气显示**: 左上角实时天气信息，点击刷新
4. **音乐播放**: 右上角音乐播放器，8首歌曲随机播放
5. **鼠标特效**: 点击时金色特效，Konami彩蛋
6. **功能导航**: 点击卡片进入AI聊天或软件下载

### AI聊天使用
1. 点击"AI智能聊天"进入聊天页面
2. 选择AI模型（GPT-4o、DeepSeek等）
3. 开始对话，支持多标签切换
4. 遇到API限制时可切换模型
5. 支持中英文界面切换

### 软件下载使用
1. 点击"资源下载中心"进入软件页面
2. 选择分类查看不同软件
3. 点击"下载"按钮直接下载
4. 媒体文件支持预览功能
5. 右上角可进入隐藏功能

### 小游戏使用
1. 在软件下载页面点击"🎮 小游戏"
2. 选择想要玩的游戏
3. 点击"开始游戏"开始
4. 查看游戏说明和操作方式
5. 支持键盘和触摸控制

### 隐藏功能使用
1. 在软件下载页面找到"🔒 隐藏入口"
2. 输入正确暗号（小写英文）
3. 进入隐藏页面观看视频
4. 3秒后自动返回主页

## 🎨 自定义

### 添加新软件
在 `software-library.html` 中的 `softwareData` 数组添加：
```javascript
{
    id: "新软件ID",
    name: "软件名称",
    category: "Windows应用",
    description: "软件描述",
    size: "文件大小",
    downloadUrl: "下载链接"
}
```

### 添加新游戏
1. 创建新的游戏HTML文件
2. 在 `game-page.html` 中添加游戏卡片
3. 实现游戏逻辑和界面

### 修改样式
- 主要样式在HTML文件的 `<style>` 标签中
- 支持CSS3高级特性
- 响应式设计适配不同设备

### 添加新音乐
在 `index.html` 中的 `songs` 数组添加：
```javascript
{
    url: '音乐链接',
    name: '歌曲名称'
}
```
